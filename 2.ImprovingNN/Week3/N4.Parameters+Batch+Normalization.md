### 🧬 Comparativa de Betas

| Contexto | Tipo | Función |
| :--- | :--- | :--- |
| **Momentum / Adam** | Hiperparámetro | Controlar la "memoria" del gradiente. |
| **Batch Norm ($\beta$)** | Parámetro ($b$) | **Desplazar** los datos (Shift). |
| **Batch Norm ($\gamma$)** | Parámetro ($W$) | **Escalar** los datos (Scale/Stretch). |

### 🔬 La fórmula simplificada
$$Z_{final} = \gamma \cdot Z_{norm} + \beta$$

> [!TIP]
> Estos parámetros existen para que la red neuronal tenga la **libertad** de deshacer la normalización si descubre que los datos originales eran mejores.
